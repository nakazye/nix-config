# ユーザー設定

このファイルはすべてのClaude Codeセッションに適用されるユーザーレベルの設定です。

## 言語

- **IMPORTANT**: 日本語で応答すること
- コード内のコメントやドキュメントも日本語で記述

## 作業スタイル

- 変更は最小限に留め、要求された内容のみ実装する
- 過度な抽象化や将来の拡張性のための設計は避ける
- 既存のコードスタイルに従う
- 要件が不明確な場合は、推測せず確認する

## セキュリティ

- 機密情報（APIキー、パスワード等）をコードに含めない

## コミット

- コミットメッセージは日本語で記述
- 変更の「なぜ」を重視した簡潔なメッセージ
- 1行目は50文字以内で要約
- **IMPORTANT**: `Co-Authored-By` などのAI生成を示す署名は追加しない

## 環境

- シェル: Zsh
- パッケージ管理: Nix
  - **IMPORTANT**: インストールされていないツールは `nix run nixpkgs#<package>` で一時的に利用する
